# READ ME
This repository contains validated, version-controlled Beckman Coulter Biomek i7 liquid handling methods developed by Minderoo OceanOmics Centre to process samples through eDNA and whole-genome sequencing workflows.  

# INSTRUMENT FILES
This folder contains Instrument Files (bif) for two separate Beckman Coulter Biomek i7 liquid handling robots located at the Minderoo OceanOmics Centre. Each Biomek i7 has a slightly different deck configuration, reflected in their individual instrument files.

# INSTRUCTIONS FOR USE
Each Biomek method has an associated laboratory procedure published on protocol IO **https://www.protocols.io/researchers/oceanomics-uwa** . Please refer to the appropriate protocol for each method.

## Notes

Users should review the method, deck layout, labware definitions and instrument setup before use. The method should be simulated and tested with water before use with biological samples, valuable reagents or production workflows.

## Disclaimer

These files are provided for documentation and research-use purposes only. They are shared **as is**, without warranty or guarantee of performance on other instruments, configurations or workflows.

Use of these files is at the user’s own risk. Users are responsible for confirming compatibility with their own instrument, software version, deck configuration, labware, reagents, safety requirements and institutional procedures before running any method.
